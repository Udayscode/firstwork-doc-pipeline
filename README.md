# 📄 FirstWork Document Understanding Pipeline

An AI-powered pipeline that extracts structured data from Driving Licenses, Shop Receipts, and Resumes using Google Gemini and OCR tools.

## 🚀 Features
- Prompt-based LLM extraction (Gemini Pro)
- OCR support via Tesseract
- Modular, scalable architecture
- Handles noisy docs and new fields
- JSON outputs with fallback validation
